# 🐍 Data Science Journey — Python

A collection of Jupyter notebooks documenting my journey learning Python for data science — covering core Python fundamentals, pattern-based problem solving, and the essential data science libraries: NumPy, Matplotlib, and Seaborn.

## 📚 Contents

| Notebook | Description |
|---|---|
| [`1_Python.ipynb`](./1_Python.ipynb) | Core Python concepts and fundamentals — syntax, data types, control flow, functions, and more. |
| [`2_pattern_question.ipynb`](./2_pattern_question.ipynb) | Practice problems on printing patterns (stars, numbers, pyramids, etc.) using loops. |
| [`3_numpy.ipynb`](./3_numpy.ipynb) | Introduction to NumPy — arrays, indexing/slicing, array operations, and broadcasting. |
| [`numpy.ipynb`](./numpy.ipynb) | Additional NumPy exercises and examples. |
| [`5_Matplotlib.ipynb`](./5_Matplotlib.ipynb) | Data visualization with Matplotlib — line plots, bar charts, histograms, and plot customization. |
| [`6_Seaborn.ipynb`](./6_Seaborn.ipynb) | Statistical data visualization with Seaborn — distribution plots, categorical plots, and styling. |

## 🎯 Purpose

This repository serves as a personal learning log and reference while building a foundation in Python programming and the data science stack. Each notebook is self-contained and focuses on a specific topic, with code examples and short explanations along the way.

## 🚀 Getting Started

### Prerequisites

Make sure you have Python 3 and Jupyter installed. You can install the required libraries with:

```bash
pip install numpy matplotlib seaborn jupyter
```

### Running the notebooks

1. Clone the repository:
   ```bash
   git clone https://github.com/ManishaDutt11/python.git
   cd python
   ```

2. Launch Jupyter Notebook:
   ```bash
   jupyter notebook
   ```

3. Open any `.ipynb` file from the browser interface and run the cells.
